## Install

### 1. Prepare Environment

* [Node](https://nodejs.org)
* [Hexo](https://hexo.io)

You can visit these offical websites to install required components.

### 2. Clone Project

```
git clone https://github.com/itren/itren.github.io.git
```

### 3. Install Dependencies

```
cd itren.github.io
npm install
```

## Deploy

### 1. Generate Website

```
hexo g
```

### 2. Starting Server

```
hexo s
```
